Incremental Ready

增量同步流程

当前只固定站点本体，下一步再让 Codex Skill 适配这个结构。
01

生成论文页

每篇论文写入 papers/<slug>/index.html，同目录可放图片和表格资源。

02

更新 JSON 索引

向 data/papers.json 新增或更新一条记录，主页自动进入搜索、分类、标签和索引。

03

提交到 GitHub

执行 git add .、git commit、git push 后，GitHub Pages 自动刷新。

04

持续迭代

同一个 slug 可反复更新：粗读版 → 精读版 → 复现版，不影响首页结构。
